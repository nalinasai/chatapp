**ChatApp – Java Socket Chat Application**
A simple two-way chat application built using Java Sockets and AWT GUI.
This project contains:

**Server Application → nalina.java
Client Application → friend.java**

Both programs communicate through a TCP connection and exchange messages in real time.


**Features**
Real-time messaging between two laptops
Simple AWT-based GUI (TextField, TextArea, Button)
Uses ServerSocket and Socket for communication
Multi-threaded message receiving
Clean and lightweight code
Works on any device that supports Java
